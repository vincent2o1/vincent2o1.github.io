BrainX

BrainX is a brain-inspired AI project that connects perception, memory, emotion, damage response, imagination, and Unity simulation.

Architecture

Perception

BrainX receives visual and hearing inputs and converts them into internal representations.

Memory

The ring-tree memory system stores patterns, compares new inputs, and retrieves related past experiences.

Emotion and Reflex

Emotional centers detect negative states, store recovery patterns, and create reflex-like responses.

Unity Simulation

The Unity agent receives commands from Python, moves inside a simulation, detects damage, and sends feedback back to BrainX.
